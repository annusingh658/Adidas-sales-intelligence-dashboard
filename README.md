## Overview

The Adidas Sales Intelligence Dashboard is an interactive business intelligence and analytics application developed using SQL, Python, Pandas, Streamlit, and Plotly.

The project follows a database-driven analytics workflow where sales data is stored in a SQLite database, retrieved using SQL queries, processed using Pandas, and visualized through interactive dashboards.

The dashboard transforms raw Adidas sales data into actionable business insights, enabling users to analyze sales performance, retailer contributions, regional growth, and business trends through an intuitive interface.

## 📊 Dashboard Features

### Retailer Performance Analysis

Analyze and compare sales generated by different retailers using interactive bar charts.

### Monthly Sales Trend Analysis

Track sales performance over time and identify growth patterns using dynamic trend visualizations.

### State-wise Performance Dashboard

Compare total sales and units sold across states using dual-axis analytics.

### Regional Sales Intelligence

Explore sales contribution across regions and cities using hierarchical treemap visualizations.

### Interactive Data Exploration

View and explore processed business data directly from the dashboard.

### Report Download Functionality

Export analytical reports and datasets for further business analysis.
